# Исследование современных оптимизационных алгоритмов в задачах управления техническими системами
В работе исследуются пять метаэвристических алгоритмов на трёх классических задачах теории управления. Основное внимание уделяется:
- **Точности** найденных решений (значение целевой функции)
- **Вычислительной эффективности** (время выполнения, количество вызовов функции)
- **Стабильности** (стандартное отклонение результатов при множественных запусках)
- ## Установка и запуск

### Предварительные требования
- **Python 3.8**
- **Git**

### Необходимые библиотеки
numpy, scipy, matplotlib, pandas, control, psutil, но также зависимости прописаны в файле requirements.txt
# Установить все зависимости:
pip install -r requirements.txt
# Запустить эксперименты
python experiment.py
